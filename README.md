# BatmanBuilt

BatmanBuilt is a job-site project tracker with photos, documents, field notes, and construction calculators.

## Deploy Free On Vercel

1. Create a free GitHub account.
2. Create a free Vercel account using **Continue with GitHub**.
3. Put the contents of this folder in a GitHub repository.
4. In Vercel, import that GitHub repository.
5. Use the default static-site settings. No build command is required.

Vercel will create a free live URL like:

`https://your-project-name.vercel.app`

## Important Data Note

This version stores projects, photos, PDFs, and calculator data in the browser using localStorage and IndexedDB.
That means the site can load from any phone or computer, but saved project data does not automatically sync between devices yet.
